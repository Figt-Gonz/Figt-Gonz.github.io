# 心游的技术博客

基于 [AstroPaper](https://github.com/satnaing/astro-paper) 搭建的个人技术博客，部署到 GitHub Pages。

## 本地开发

```bash
npm install
npm run dev
```

## 构建验证

```bash
npm run build
```

## 部署

推送到 `main` 分支后，GitHub Actions 会自动构建并部署到 GitHub Pages。

线上地址：<https://Figt-Gonz.github.io/>

## 内容方向

- Android 技术实践
- AI 工具使用经验
- 工程效率优化
- 职业成长思考
- 公众号文章归档
